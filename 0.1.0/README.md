# SJTU Master Midterm Examination Report Typst Template
# 上海交通大学硕士研究生学位论文中期检查报告 Typst 模板

本项目是 **上海交通大学硕士研究生学位论文中期检查报告** 的 Typst 模板。本模板严格根据上海交通大学研究生院官网提供的最新版 [Word 模板](https://www.gs.sjtu.edu.cn/xzzx/pygl) 要求制作。

相比于传统的 LaTeX 模板，本模板具有 **极速编译**（毫秒级响应）以及 **零环境配置** 等优势。

## 使用方法

### Typst Web App (网页版)

这是最简单、免安装的使用方式：

1. 登录 [Typst Web App](https://typst.app/)。
2. 点击 **"Start from template"** 按钮。
3. 在搜索框输入 `sjtu-master-midterm-report`。
4. 在结果中点击本模板并点击 **"Create"**，系统将自动根据 `main.typ` 初始化工程，您可以直接开始撰写正文。


## 字体要求

为了达到最佳排版效果，请确保您的环境（或系统）已安装以下字体：

* **中文字体**：宋体 (SimSun)、仿宋 (FangSong)、楷体 (KaiTi)、黑体 (SimHei)。
* **英文字体**：Times New Roman。
